# replace_xlsx
An application that inserts text and images into xlsx cells

# Features
Inspired by Apache POI and OpenXML

# Requirement
See Cargo.toml.

# Usage

```bash
cargo run -- -t tests/in.xlsx -o out.xlsx < tests/input.json
```
